# Anti-PM Telegram Userbot

## Setup
- Install dependencies: `pip install -r requirements.txt`
- Set Environment Variables:
  - API_ID
  - API_HASH
  - SESSION_NAME

## Commands
- /approve
- /disapproval
- /unblock
- /listapproved
- /help

## Features
- 3 Warnings then block
- Approve/unblock system
- Permanent Railway Hosting Supported
